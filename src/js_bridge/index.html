WebMSX JS Bridge Samples

WebMSX と JavaScript を I/O ポート経由でつなぐためのサンプル集です。以下のページから、目的に合う ブリッジ実装をすぐ試せます。

IO Port JS Bridge Sample 1

BASIC と JavaScript をポート 0x50〜0x53 で連携させる最小構成の例です。キーボー ド入力で OUT を送り、ページ内のボタン操作で IN を返す動作を確認できます。

MP3 Bridge Sample

MSX 側のコマンドでホストブラウザーの MP3 再生を制御するサンプルです。0x5A と 0x5B の I/O ポートを使って BGM や SE の再生、フェード、クロスフェードを行います。

デモを開く スクリプトを見る
リポジトリーのルートでサーバーを起動し、http://localhost:8000/src/js_bridge/index.html を開くとこの一覧を表示できます。
