IO Port JS Bridge Sample 1

WebMSX 本体と bridge.js をまとめて読み込むデモページです。ローカルでページを開くだけで サンプル JavaScript が動作するので、エミュレーターの I/O ポート連携をすぐ確認できます。

使い方

  1. リポジトリーのルートで簡易サーバーを起動します（例: python -m http.server）。
  2. ブラウザーで http://localhost:8000/src/js_bridge/sample1/index.html を開きます。
  3. エミュレーターの下に表示されるパネルで OUT/IN の状態を確認できます（自動で接続されます）。
  4. 下の BASIC リストを WebMSX にペーストして RUN すると、キー入力と JS 側の連携を試せます。
